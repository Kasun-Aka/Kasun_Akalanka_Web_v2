# KasunAkalanka_Web_v2

# 🌐 Personal Portfolio Website — KasunAkalanka | GhostDevKA

A modern, fully functional **personal website** built to showcase my work, skills, and journey as a software engineering student.  
Unlike a typical portfolio, this project includes **backend functionality, authentication, messaging, and dynamic content**, making it a complete full-stack application (v2).

🔗 **Live Website:** https://kasun-akalanka-web.vercel.app/  

---

## 🚀 Features

### 🎨 Frontend (React • TypeScript • Vite)
- **About Me** section with personal introduction  
- **Skills** with categorized technical stacks  
- **My Projects** with clean layouts  
- **Contact Me** section with dynamic form handling  
- Fully responsive UI  
- Smooth animations (AOS) & typing effects (Typed.js)

---

## 🛠️ Backend (Node.js • Express • Mongoose)
- **Firebase Authentication**  
  - Secure sign-up & sign-in    
- **MongoDB Database**  
  - Stores contact messages  
  - Stores user details  
- **API Endpoints** for sending messages, authentication, and data handling  
- Advanced Security with Helmet, Rate Limiting & Mongo Data Sanitization

---

## 📦 Tech Stack

### **Frontend**
- React 19
- TypeScript
- Vite
- React Router DOM

### **Backend**
- Node.js  
- Express.js  

### **Database**
- MongoDB (via Mongoose)

### **Auth**
- Firebase Authentication & Firebase Admin SDK

### **Hosting**
- Vercel (Frontend) & Render (Backend)
